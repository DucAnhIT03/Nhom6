# car-booking-admin-final

React + Vite + Tailwind admin dashboard (desktop) for Car Booking project.

## Setup

1. Ensure Node >= 18
2. unzip and cd into project
3. npm install
4. npm run dev

Pages:
- /admin
- /admin/users
- /admin/stations
- /admin/companies
- /admin/buses
- /admin/bus-station
- /admin/routes
- /admin/posts

Data is mocked in src/services/*.js for frontend testing.
